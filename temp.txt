Phase 1 : Server Setup
1. app.listen()
Syntax
app.listen(PORT, () => {
    console.log(`Server running on port ${PORT}`);
});
Meaning

Server ko start karta hai aur specified port par requests sunna shuru karta hai.

Why are we using it?

Without it, hamara Express application start hi nahi hoga.

Internal Working
Client
↓
Port 5000
↓
Express Server
Real World Analogy

Restaurant ka shutter kholna.
